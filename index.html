Verbs

Verb Description ID (IRI) Vocabulary
answered Indicates the actor replied to a question, where the object is generally an activity representing the question. The text of the answer will often be included in the response inside result. https://w3id.org/xapi/adl/verbs/answered https://w3id.org/xapi/adl
attempted Indicates the actor made an effort to access the object. An attempt statement without additional activities could be considered incomplete in some cases. https://w3id.org/xapi/adl/verbs/attempted https://w3id.org/xapi/adl
completed Indicates that the actor has completed the object. https://w3id.org/xapi/adl/verbs/completed https://w3id.org/xapi/adl
loggedin Indicates the actor gained access to a system or service by identifying and authenticating with the credentials provided by the actor. https://w3id.org/xapi/adl/verbs/logged-in https://w3id.org/xapi/adl
clicked Indicates the actor clicked the object. https://w3id.org/xapi/let/verbs/clicked https://w3id.org/xapi/let
lunched Indicates the actor attempted to start an activity. https://w3id.org/xapi/adl/verbs/lunched https://w3id.org/xapi/adl

Activity Types

Acitivity Type Description ID (IRI) Vocabulary
assessment This term is used for assessments in digital form, to be distinguished from assessment in paper form. https://w3id.org/xapi/adl/activities/assessment https://w3id.org/xapi/adl
question A question is typically part of an assessment and requires a response from the learner, a response that is then evaluated for correctness. https://w3id.org/xapi/adl/activities/question https://w3id.org/xapi/adl
application Represents any kind of software application. http://activitystrea.ms/schema/1.0/application http://activitystrea.ms/schema/1.0/
resource A resource is a generic item that the actor may use for something. It could be a video, a text article, a device, etc. However, it is recommended to use resourceType extension. https://w3id.org/xapi/let/activities/resource https://w3id.org/xapi/let

Extensions

Extension Description ID (IRI) Vocabulary
subType The subType extension should be used to identify the system type that is being logged into. http://xapi.jisc.ac.uk/subType http://xapi.jisc.ac.uk/
resourceType The subType extension should be used to identify the system type that is being logged into. https://w3id.org/xapi/let/Extensions/resource https://w3id.org/xapi/let/
